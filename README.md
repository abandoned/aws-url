# AWS URL

[![travis][status]][travis]

AWS URL is a minimum-viable URL builder for [Amazon Web Services (AWS)][aws] API queries. It signs requests with [Signature Version 2][sign].

![bezos][bezos]

[Vacuum][vacuum] and [Peddler][peddler] use AWS URL.

[Fog][fog] doesn't.
